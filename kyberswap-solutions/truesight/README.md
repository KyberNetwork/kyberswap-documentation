# KyberAI

## Overview

KyberAI was conceived with the intention of surfacing valuable market data that would empower KyberSwap users to make data-driven trades. KyberAI aggregates both on-chain and off-chain data across [multiple chains](../../getting-started/supported-exchanges-and-networks.md) and condenses this wealth of data into actionable insights which can be conveniently accessed via the [KyberSwap Interface](https://kyberswap.com/discover). With KyberAI, you can ape smartly into the next big opportunity.
